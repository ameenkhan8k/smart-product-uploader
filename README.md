# Smart Product Uploader & SEO Optimizer

This n8n workflow automates the process of reading product info from Google Sheets, using OpenAI to generate SEO content, and uploading it to Notion.

## Features
- Google Sheets integration
- OpenAI-powered SEO content
- Auto-push to Notion databases

## Tech Stack
- n8n
- Google Sheets
- OpenAI GPT
- Notion API
